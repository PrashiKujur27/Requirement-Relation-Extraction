Context-Aware Requirement Relation Extraction using Transformer Models (BERT/RoBERTa) for detecting conflict, support, and neutral relations among software requirements. Includes preprocessing, fine-tuning, evaluation (Precision/F1), and relation graph visualization using NetworkX.
